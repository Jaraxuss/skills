# 章节页风格总结

## 概述

章节页（Section/Chapter Page）用于作为演示内容的分隔和过渡页面，采用**极简设计**，突出单一核心信息，营造视觉冲击力。

---

## 1. 设计规范

### 1.1 画布与尺寸
| 属性 | 值 | Tailwind 类 |
|------|-----|-------------|
| 画布尺寸 | 1280 × 720 px | `w-[1280px] h-[720px]` |
| 内边距 | 48px | `p-12` |

### 1.2 颜色系统
| 用途 | 颜色值 | CSS 变量 |
|------|--------|----------|
| 品牌主色 | #C00000 | `--rpa-red` |
| 背景色 | #F8F9FA | `--bg-light` |
| 外框背景 | 深灰/黑 (#1F2937) | `bg-gray-900` |
| 主标题色 | #1F2937 | `text-gray-900` |
| 副标题色 | #6B7280 | `text-gray-500` |
| 装饰线色 | 浅红 (#FEE2E2) | `bg-red-100` |

### 1.3 字体系统
| 用途 | 字体栈 |
|------|--------|
| 西文 | Inter |
| 中文 | Noto Sans SC |
| 等宽 | JetBrains Mono |

```css
.font-sans {
    font-family: 'Inter', 'Noto Sans SC', sans-serif;
}
```

## 2. 布局模板

### 2.1 页面结构

```
┌─────────────────────────────────────────┐
│  外框 (bg-gray-900, 全屏居中)             │
│  ┌───────────────────────────────────┐  │
│  │  画布 (1280x720, bg-[#F8F9FA])     │  │
│  │                                   │  │
│  │     ┌─────────────────────┐       │  │
│  │     │    红色光晕装饰       │       │  │
│  │     └─────────────────────┘       │  │
│  │                                   │  │
│  │     ┌─────────────────────┐       │  │
│  │     │   超大主标题          │       │  │
│  │     │   ─────────         │       │  │
│  │     │   装饰线（居中）       │       │  │
│  │     │                     │       │  │
│  │     │   副标题              │       │  │
│  │     └─────────────────────┘       │  │
│  │                 ↑ 垂直水平居中      │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

### 2.2 核心布局特点

1. **完全居中**：内容区域垂直水平完全居中
2. **极简元素**：仅包含标题 + 装饰线 + 副标题
3. **层次分明**：通过字号、颜色、间距区分层级

## 3. 组件详解

### 3.1 主标题

| 属性 | 值 | Tailwind 类 |
|------|-----|-------------|
| 字号 | 120-160px | `text-[120px]` ~ `text-[160px]` |
| 字重 | Black (900) | `font-black` 或 `font-extrabold` |
| 颜色 | 深灰 | `text-gray-900` 或 `text-[#1F2937]` |
| 字间距 | 紧凑 | `tracking-tight` |
| 行高 | 无额外行高 | `leading-none` |
| 底部间距 | 24-32px | `mb-6` ~ `mb-8` |

### 3.2 装饰线

| 属性 | 值 | Tailwind 类 |
|------|-----|-------------|
| 宽度 | 96-128px | `w-24` ~ `w-32` |
| 高度 | 4-8px | `h-1` ~ `h-2` |
| 颜色 | 浅红色 / 品牌红 | `bg-red-100` 或 `bg-[#C00000]` |
| 圆角 | 完全圆角 | `rounded-full` |
| 底部间距 | 32-40px | `mb-8` ~ `mb-10` |
| 居中 | 水平居中 | `mx-auto` |

### 3.3 副标题

| 属性 | 值 | Tailwind 类 |
|------|-----|-------------|
| 字号 | 36px | `text-4xl` |
| 字重 | Medium | `font-medium` |
| 颜色 | 中灰 | `text-gray-500` |
| 字间距 | 宽松 | `tracking-widest` |

### 3.4 背景光晕

| 属性 | 值 | Tailwind 类 |
|------|-----|-------------|
| 尺寸 | 500-800px | `w-[500px] h-[500px]` |
| 位置 | 左上角偏移 | `-top-20 -left-20` |
| 颜色 | 红色 5% 透明度 | `from-red-500/5` 或 `from-[#C00000]/5` |
| 形状 | 圆形 | `rounded-full` |
| 模糊 | 大范围 | `blur-3xl` |
| 渐变方向 | 向右下 | `bg-gradient-to-br` |

## 4. 动画规范

### 4.1 入场动画

```css
@keyframes fade-in-up {
    0% {
        opacity: 0;
        transform: translateY(20px);
    }
    100% {
        opacity: 1;
        transform: translateY(0);
    }
}

.animate-enter {
    animation: fade-in-up 0.8s ease-out forwards;
}
```

### 4.2 动画延迟序列

| 元素 | 延迟时间 |
|------|----------|
| 主标题 | 0s |
| 装饰线 | 0.1s |
| 副标题 | 0.2s |

## 5. 固化 HTML 片段

### 5.1 完整章节页模板

```html
<!DOCTYPE html>
<html lang="zh">

<head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>{{章节标题}}</title>
    <script src="../../lib/tailwind@4.js"></script>
    <link href="../../lib/font-awesome/all.min.css" rel="stylesheet" />
    <link href="../../lib/fonts.css" rel="stylesheet" />
    <style>
        /* 字体设置 */
        .font-sans {
            font-family: 'Inter', 'Noto Sans SC', sans-serif;
        }

        /* 隐藏滚动条 */
        ::-webkit-scrollbar {
            display: none;
        }

        /* 影刀品牌色 */
        .text-rpa-red {
            color: #C00000;
        }

        .bg-rpa-red {
            background-color: #C00000;
        }

        /* 入场动画 */
        @keyframes fade-in-up {
            0% {
                opacity: 0;
                transform: translateY(20px);
            }
            100% {
                opacity: 1;
                transform: translateY(0);
            }
        }

        .animate-enter {
            animation: fade-in-up 0.8s ease-out forwards;
        }

        /* 脉冲动画（可选） */
        @keyframes pulse-subtle {
            0%, 100% {
                opacity: 0.8;
            }
            50% {
                opacity: 1;
            }
        }

        .animate-pulse-subtle {
            animation: pulse-subtle 4s ease-in-out infinite;
        }
    </style>
</head>

<body
    class="bg-gray-900 flex items-center justify-center h-screen w-screen selection:bg-red-100 selection:text-red-800">
    <div id="page-shell" class="flex items-center justify-center h-full w-full">
        <div id="slide-scale-wrapper" class="relative w-[1280px] h-[720px]">

            <!-- 画布容器 -->
            <div id="slide-container"
                class="w-full h-full box-border flex flex-col relative overflow-hidden bg-[#F8F9FA] p-12">

                <!-- 背景装饰：红色光晕 -->
                <div
                    class="absolute -top-20 -left-20 w-[500px] h-[500px] bg-gradient-to-br from-red-500/5 to-transparent rounded-full blur-3xl pointer-events-none">
                </div>

                <!-- 主要内容区：居中展示 -->
                <div class="flex-1 flex flex-col items-center justify-center z-10 relative">

                    <!-- 主标题 -->
                    <h1 class="text-[120px] font-black text-[#1F2937] tracking-tight leading-none mb-6 animate-enter">
                        {{章节标题}}
                    </h1>

                    <!-- 装饰线 -->
                    <div class="w-24 h-1 bg-[#C00000] rounded-full mb-8 animate-enter"
                        style="animation-delay: 0.1s;"></div>

                    <!-- 副标题 (可选) -->
                    <p class="text-4xl text-gray-500 font-medium tracking-widest animate-enter"
                        style="animation-delay: 0.2s;">
                        {{副标题文字}}
                    </p>

                </div>

            </div>
        </div>
    </div>

    <script src="../../js/slide-core.js"></script>
</body>

</html>
```

## 6. 使用指南

### 6.1 适用场景

- 演示文稿的章节分隔页
- 内容过渡页（如"开始"、"Q&A"、"感谢"等）
- 需要强调单一信息的页面

### 6.2 替换占位符

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `{{章节标题}}` | 主要章节名称 | "小试影刀"、"Q&A" |
| `{{副标题文字}}` | 补充说明或英文翻译 | "GETTING STARTED" |

### 6.3 样式变体

1. **超大标题**：`text-[160px]` 适用于短文字
2. **标准标题**：`text-[120px]` 适用于一般章节
3. **带品牌色装饰线**：`bg-[#C00000]` 更醒目
4. **柔和装饰线**：`bg-red-100` 更低调
