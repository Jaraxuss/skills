# 标准含视频详情页风格总结

## 概述

标准含视频详情页用于展示 RPA 解决方案的详细介绍，采用**左右分栏布局**：左侧为功能说明卡片区，右侧为视频播放器。整体设计专业、现代，具有强烈的企业级产品感。

---

## 1. 设计规范

### 1.1 画布与尺寸
| 属性 | 值 | Tailwind 类 |
|------|-----|-------------|
| 画布尺寸 | 1280 × 720 px | `w-[1280px] h-[720px]` |
| 内边距 | 48px | `p-12` |
| 左右分栏间距 | 40px | `gap-10` |

### 1.2 颜色系统
| 用途 | 颜色值 | Tailwind 类 |
|------|--------|-------------|
| 品牌主色 | #C00000 | `.text-rpa-red` / `.bg-rpa-red` |
| 背景色 | #F8F9FA | `bg-[#F8F9FA]` |
| 外框背景 | #1F2937 | `bg-gray-900` |
| 卡片背景 | #FFFFFF | `bg-white` |
| 价值条背景 | #1F2937 | `bg-gray-800` |
| 主标题色 | #111827 | `text-gray-900` |
| 副标题色 | #6B7280 | `text-gray-500` |

### 1.3 图标颜色配色方案
| 功能类型 | 图标背景色 | 图标颜色 |
|----------|------------|----------|
| 交互类 | `bg-red-50` | `text-rpa-red` |

| 时间类 | `bg-yellow-50` | `text-yellow-400` |
