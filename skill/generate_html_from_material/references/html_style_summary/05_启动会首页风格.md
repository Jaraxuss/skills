# 启动会首页 风格总结

## 概述

用于作为整个演示文稿的首页，采用现代企业级设计风格，以中心对称布局为主，通过大标题、副标题和底部特性卡片组合呈现项目核心信息。整体风格简洁专业，融合玻璃拟态（Glassmorphism）设计和微妙的光晕效果，营造高端科技感。

## 分析的页面

- `./pages/启动会首页.html`

---

## 1. 设计规范

### 画布尺寸
- **固定尺寸**: `1280px × 720px` (16:9 演示比例)
- **缩放容器**: `#slide-scale-wrapper` 设置固定宽高，外层 `#page-shell` 居中显示

### 背景设计
- **基础背景**: `#F8F9FA` (浅灰色)
- **装饰光晕**:
  - 右上角红色光晕: `#C00000/5` (5% 不透明度)，尺寸 `600×600px`，高斯模糊 `blur-3xl`
  - 左下角蓝色光晕: `#3370FF/5` (飞书蓝)，尺寸 `800×800px`，高斯模糊 `blur-3xl`
- **网格纹理**: 径向渐变点阵图案，`40px × 40px` 间距，透明度 30%

### 品牌色
- **主色 - 影刀红**: `#C00000`
- **辅助色 - 飞书蓝**: `#3370FF`
- **中性色**:
  - 深灰标题: `#1F2937`
  - 次级标题: `#374151`
  - 正文灰: `#4B5563`
  - 浅色文字: `#6B7280`

### 字体规范
- **字体族**: `'Inter', 'Noto Sans SC', sans-serif`
- **标题层级**:
  - 主标题: `76px`, `font-bold`, `leading-[1.15]`
  - 副标题: `64px`, `font-bold`
  - Slogan: `24px`, `font-medium`
  - 卡片标题: `18px`, `font-bold`
  - 卡片描述: `12px`, `text-xs`

## 4. CSS 类名与样式约定

### Tailwind CSS 应用
- **颜色系统**: 使用 Tailwind 内置色板 + 自定义品牌色
- **间距**: 使用 Tailwind spacing scale (`p-6`, `gap-5`, `px-20` 等)
- **圆角**: `rounded-2xl` (16px), `rounded-xl` (12px), `rounded-full` (9999px)
- **阴影**: 使用 Tailwind 内置 shadow utilities

### 自定义类名
```css
.glass-card {
  /* 玻璃拟态卡片 */
```

## 5. 动画约定

### 入场动画
**关键帧定义**:
```css
@keyframes fade-in-up {
  0% {
    opacity: 0;
    transform: translateY(20px);
  }
  100% {
    opacity: 1;
    transform: translateY(0);
  }
}
```

**应用类**:
```css
.animate-enter {
  animation: fade-in-up 0.8s ease-out forwards;
}
```

**延迟规则**:
- 主内容区: 无延迟 (立即播放)
- 底部卡片组: `animation-delay: 0.2s`

### 交互动画
**玻璃卡片悬停**:
```css
transition: all 0.3s ease;

/* 悬停效果 */
transform: translateY(-5px);
background: rgba(255, 255, 255, 0.8);
box-shadow: 增强;
```

**图标缩放**:
```css
group-hover:scale-110;
transition: transform 0.3s;
```

**联系信息渐显**:
```css
opacity: 0.6 → 1;
transition: opacity 0.5s;

/* 图标滑入 */
translate-x: 8px → 0;
opacity: 0 → 1;
```

## 使用说明

### 变量替换
- `{{CLIENT_NAME}}`: 客户名称占位符，生成时替换为实际客户名
- 联系人信息: 根据实际需求修改姓名、角色和电话

### 核心特性卡片自定义
可根据项目特点替换三张卡片的内容：
1. 修改图标 SVG
2. 更新标题和描述文字
3. 调整配色方案（保持橙/红/蓝的视觉平衡）

### 品牌色适配
如需使用其他品牌色，替换以下颜色值：
- `#C00000` → 新的主品牌色
- `#3370FF` → 新的辅助品牌色
- 同步更新光晕、标签、卡片等相关组件的颜色

### 响应式处理
当前版本为固定尺寸演示页面，通过 `slide-core.js` 实现缩放适配。如需响应式布局，需要：
1. 将固定像素值改为相对单位
2. 添加媒体查询
3. 调整布局方式为流式布局
